# Tourify Luxury Travel

Welcome to the **Tourify** repository – a premium, bilingual (Arabic & English) travel agency platform designed to deliver white-glove, elite travel setups, from majestic Saharan expeditions across Djanet and Tassili N'Ajjer to bespoke Umrah packages and international destination access.

## Features & Highlights

- 🌍 **Dynamic Routing Architecture**: Fully modernized with React Router (v7) for secure deep-linking and seamless SPA navigation.
- 💎 **Elite Visual Design**: A dark, atmospheric UI with luxury-themed `vibrant gradients`, `glassmorphism`, and exclusive aesthetics utilizing top-tier Tailwind CSS patterns.
- 🌓 **Full Translation Support**: Native dynamic flipping between Left-to-Right (English) and Right-to-Left (Arabic), retaining perfect style consistency.
- 🏨 **Destination Discovery**: Rich itineraries mapping exclusive activities and tailored exploration packages.
- 📝 **Live Inquiry System**: Smooth coordinates registry allowing elite customers to reach out on WhatsApp or email transparently. 
- 🚀 **Next-Gen Tech Stack**: Vite + React 18 + TypeScript + Tailwind CSS for lightning-fast delivery and robust stability.

## Development

This repository has been configured to build and deploy perfectly out of the box on services like Vercel and Netlify via automatically injected rewrite configurations.

### Install & Run

1. **Install dependencies:**
   \`\`\`bash
   npm install
   \`\`\`

2. **Start the local development server:**
   \`\`\`bash
   npm run dev
   \`\`\`

3. **Production build:**
   \`\`\`bash
   npm run build
   \`\`\`
