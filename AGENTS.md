### Icons & Visual Assets

Do NOT use free basic icon libraries like Lucide React.

Use premium-looking modern icons similar to high-end travel and SaaS websites.

Preferred icon libraries:

* Phosphor Icons Pro style
* Huge Icons
* Iconoir
* Solar Icons
* Remix Icons
* Tabler Icons (premium usage style)
* Streamline Icons
* Iconsax
* Fluent UI Icons

Icons should feel:

* Elegant
* Thin/medium stroke
* Modern
* Luxury
* Minimal
* Consistent across the entire website

Use animated icons and gradient icon containers where appropriate.

Avoid generic startup-style icons. The website should feel like a premium international travel brand, similar to luxury airline and tourism platforms.
